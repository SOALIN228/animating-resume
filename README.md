## 开发中学到的知识

1. 异步和同步

   异步就是通知你做事，但是不等结果，直接下一步

   同步就是通知你做事，等你做完，拿到结果，再执行下一步

2. 异步和回调

   异步中经常和回调联系在一起，因为通过回调可以拿到异步的结果
